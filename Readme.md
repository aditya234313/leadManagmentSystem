# Mini CRM - Lead Management System

## Tech Stack
- React.js
- Node.js
- Express.js
- PostgreSQL

## Features
- Add Leads
- Update Lead Status
- Delete Leads
- REST APIs
- PostgreSQL Integration

## Setup Instructions

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm run dev

## API Endpoints

GET /api/leads
POST /api/leads
PUT /api/leads/:id
DELETE /api/leads/:id
